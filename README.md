# Pensioners Management Console Application

Консольное приложение для учета пенсионеров и их трудового стажа.

## Описание
Проект разрабатывается в рамках лабораторной работы №4. 
Реализует функционал для ведения базы данных пенсионеров, 
учета их пенсий и предыдущих мест работы.
