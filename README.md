# Fleet Management Console Application

Консольное приложение для управления автопарком и коммерческими сделками.

## Описание
Проект разрабатывается в рамках лабораторной работы №4. 
Объединяет функционал для учета грузовых рейсов (Вариант 15) 
и сделок на парфюмерном базаре (Вариант 20).
